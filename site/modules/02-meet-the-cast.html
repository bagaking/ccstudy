Display Layer Loads on Demand

`source/claude-code-source/src/replLauncher.tsx` (lines 12-21)

CODE
export async function launchRepl(...): Promise<void> {
  const { App } = await import('./components/App.js');
  const { REPL } = await import('./screens/REPL.js');
  await renderAndRun(root, <App {...appProps}><REPL {...replProps} /></App>);
}
PLAIN ENGLISH

UI shell and REPL are lazy-loaded only for interactive path.

REPL lives inside App wrapper to inherit session-wide context.

Clear ownership boundary: shell concerns vs turn-loop concerns.

Separation of concerns is operational leverage

When ownership is explicit, requests to AI can be explicit: “change QueryEngine behavior, do not touch REPL rendering.”
